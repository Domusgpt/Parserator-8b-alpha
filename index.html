Deployment checklist

Each variant references the shared assets/css/main.css and assets/js/main.js bundles, so shipping updates across builds stays synchronized.

Remove or add variants by duplicating a folder under variants/ and registering it in this index. No binaries are tracked—logos render via inline SVG for lightweight deploys.
