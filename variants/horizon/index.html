Integrate Parserator your way

Choose REST or the Node SDK—both expose telemetry and EMA safeguards.

REST endpoint

POST raw text + schema and get structured JSON with confidence scores and token telemetry.

curl -X POST "https://app-5108296280.us-central1.run.app/v1/parse" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer pk_live_your_api_key" \
  -d '{
    "inputData": "Acme Co. Contract Signed 3/12/24",
    "outputSchema": {
      "company": "string",
      "event": "string",
      "date": "date"
    }
  }'

Node SDK

Install the published SDK to manage retries, telemetry metadata, and confidence scoring with minimal setup. 

npm install parserator-sdk

import { ParseratorClient } from 'parserator-sdk';

const client = new ParseratorClient({
  apiKey: 'pk_live_your_api_key_here'
});

const result = await client.parse({
  inputData: `John Smith\nSenior Developer\njohn@techcorp.com\n(555) 123-4567`,
  outputSchema: {
    name: 'string',
    title: 'string',
    email: 'email',
    phone: 'phone'
  }
});

console.log(result.parsedData);
